# 📖 About

The **Fee Management System** is a **Java console-based application** developed using a **Singly Linked List** to manage student fee records efficiently.

It provides a menu-driven interface to perform CRUD-related operations such as adding students, updating fees, searching records, and displaying student information.

---

# ✨ Features

✅ Add Student

✅ Update Student Fees

✅ Search Student by Admission Number

✅ Display All Students

✅ Dynamic Memory Allocation

✅ Menu Driven Program
